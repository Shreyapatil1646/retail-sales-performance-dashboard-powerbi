# Retail Sales Performance Dashboard (Power BI)

## 📊 Overview
This project analyzes retail sales performance using Power BI dashboards.

## 📌 Key KPIs
- Total Sales: $2.27M
- Total Profit: $282K
- Total Orders: 5K
- Total Quantity: 37K
- Profit Margin: 12%

## 📈 Key Insights
- West region has highest sales
- Technology category leads revenue
- Some products are loss-making
- Sales trends vary over time

## 🛠 Tools Used
- Power BI
- Power Query
- DAX
